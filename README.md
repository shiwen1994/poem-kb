> 78,660 首诗词 · 5,238 位诗人 · 持续更新

## 数据规模
- 唐诗：57,607 首
- 宋词：21,053 首
- 唐诗人：3,675 位
- 宋词人：1,563 位

## 功能
- 全文搜索（诗人/诗句/典故/地点）
- 诗人关系图谱
- 典故溯源
- 诗人足迹地图（规划中）
- 历史事件时间轴（规划中）
